How to Play

  • Enter a word that starts with the highlighted letter
  • Each new word must start with the last letter of your previous word
  • No repeated words allowed in the same chain
  • Longer words = more points (length x 10)
  • Build the longest chain before time runs out!
  • Use hints if you're stuck (costs 10 points)
