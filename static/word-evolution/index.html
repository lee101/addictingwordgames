How to Play

  • Transform the start word into the target word
  • Change exactly one letter at a time
  • Each intermediate word must be a valid English word
  • Fewer steps = higher score!
  • Use hints if you're stuck (costs 15 points)

Example: CAT → COT → COG → DOG
